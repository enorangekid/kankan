열전도율 · 열관류율 · 단열재 두께의 관계

단열 설계의 핵심은 세 가지 값의 관계를 이해하는 것입니다.

기본 공식
열관류율 (U) = 열전도율 (λ) ÷ 두께 (d)
단열재 두께 (d) = 열전도율 (λ) ÷ 열관류율 (U)

열관류율 값이 작을수록 단열 성능이 높습니다. 지역별 기준값 이하를 충족하는 두께를 선택해야 합니다.

부위별 단열재 두께 계산 예시 (중부2지역 · 가등급 기준)

외벽 (공동주택 외, 외기 직접) — 열관류율 0.24 이하

0.034 ÷ 0.24 = 142mm → 실무에서는 145mm 적용 (법적 기준 135mm)

지붕·옥상 (외기 직접) — 열관류율 0.15 이하

0.034 ÷ 0.15 = 227mm → 실무에서는 230mm 적용 (법적 기준 220mm)

필로티 천장 (바닥난방·외기 직접) — 열관류율 0.17 이하

0.034 ÷ 0.17 = 200mm → 실무에서는 190~200mm 적용. 2층 주차장 상부 거실 바닥에 해당

기초 하부 바닥 (바닥난방 없음·외기 간접) — 열관류율 0.29 이하

0.034 ÷ 0.29 = 117mm → 실무에서는 110~120mm 적용

외기에 직접 면하는 경우 vs 간접 면하는 경우

외기에 직접 면하는 경우

벽·바닥·지붕의 바깥쪽이 외부에 노출되거나 외부 온도가 직접 통하는 부위. 예: 발코니 외벽, 측벽 세대의 외벽, 최상층 세대 천장 등

외기에 간접 면하는 경우

단열이 되어 있지 않은 비난방 공간과 접하는 부위. 예: 발코니와 접하는 내벽, 엘리베이터 승강로 인접 벽체, 계단실과 접하는 벽·천장 등

공동주택의 범위

아파트, 연립주택, 다세대주택, 기숙사 등이 해당합니다. 공동주택은 외기에 직접 면하는 경우 더 낮은 열관류율 기준(높은 단열 성능)이 적용됩니다.

자주 묻는 질문 (FAQ)

열관류율 기준은 왜 지역마다 다른가요?
겨울철 외기 온도 차이 때문입니다. 중부1지역(강원 내륙 등)은 기온이 낮아 더 강한 단열이 필요하므로 열관류율 기준값이 더 낮게 설정됩니다. 제주도는 온난한 기후 특성상 상대적으로 완화된 기준이 적용됩니다.
서울은 중부1지역인가요, 중부2지역인가요?
서울특별시는 중부2지역입니다. 중부1지역은 강원도 내륙, 경기 북부 일부(연천·포천·가평·남양주·의정부·양주·동두천·파주), 충청북도 제천, 경상북도 봉화·청송이 해당합니다.
단열재 두께 기준표는 어떻게 활용하나요?
시공할 지역과 부위(외벽·지붕·바닥 등), 사용할 단열재 등급(가~라)을 확인한 후 해당 두께 이상을 설치하면 됩니다. 예를 들어 서울(중부2) 아파트 외벽에 가등급 단열재를 쓴다면 190mm 이상이어야 합니다.
열관류율과 단열재 두께, 어떤 기준으로 확인해야 하나요?
인허가 설계 단계에서는 열관류율(W/㎡·K) 기준이 공식 기준입니다. 단열재 두께 기준표는 해당 두께를 충족하면 별도 계산 없이 열관류율 기준을 만족하는 것으로 인정받을 수 있는 간편 기준입니다.
2018년 이후 기준이 바뀌었나요?
본 기준표는 2018년 9월 1일 시행 기준입니다. 건축물 에너지절약설계기준은 수시로 개정될 수 있으므로 국토교통부 또는 에너지관리공단 고시를 통해 최신 기준을 반드시 확인하시기 바랍니다.
